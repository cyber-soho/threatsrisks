Lecture: Threats and Risks for Cybersecurity

1. Eradication of Threats: Techniques and Tools

Identifying and Isolating Compromised Systems

The crucial first step in eradication is to pinpoint exactly which systems have been affected by a threat and then to disconnect or segment them from the rest of the network to prevent further spread.

Network Segmentation Visualization

Techniques: Network Segmentation, Host Isolation, Firewall Rules, Process Termination.

Tools: NAC, EDR, Firewalls, Network monitoring tools (Wireshark), OS tools (Task Manager, netstat).

Examples: Isolating a malware-infected PC, blocking a compromised server's IP at the firewall, containing a ransomware attack by disconnecting file servers.

2. Preventative Measures: Best Practices

Implementing Strong Access Controls

Ensure that only authorized individuals and systems can access specific resources, based on the principle of least privilege.

Techniques: Strong Password Policies, Multi-Factor Authentication (MFA), Role-Based Access Control (RBAC).

Data Backup and Recovery Strategy

Regularly backing up critical data to secure, isolated locations ensures data can be recovered after an incident like a ransomware attack.

The 3-2-1 Backup Rule

Key Principles: 3-2-1 Rule, Immutability, Regularity, Verification.

Tools: Enterprise backup solutions (Veeam, Commvault), Cloud backup services (AWS, Azure), NAS/SAN devices.

3. Incident Response Planning

NIST Incident Response Lifecycle

A structured approach to incident management, providing a framework for handling security incidents effectively.

Lifecycle Phases

Key Elements: Creating an Incident Response Team (IRT), developing a formal Incident Response Plan (IRP), establishing communication protocols, and conducting regular tabletop exercises.

Explanations

Explanation for Exercise 1
This question tests your understanding of the "Containment" phase of incident response. The goal is to limit the damage. Disconnecting the machine from the network prevents malware from spreading (lateral movement) or communicating with its command-and-control server. Notifying the security team ensures the incident is officially tracked and managed. Avoiding personal use prevents accidental data loss or further compromise.
Explanation for Exercise 2
This question assesses your knowledge of preventative controls. A technical control is a safeguard implemented in hardware or software (like immutable backups). An administrative control is a policy or procedure that people follow (like security awareness training). A strong security posture relies on both; technology can fail, and people can make mistakes, so layering them provides resilience.
Explanation for Exercise 3
This scenario evaluates your ability to apply the Incident Response Plan. The Incident Commander's role is to coordinate the response. The Technical Lead is needed immediately to investigate and contain the breach on the server. The Communications Lead is crucial for managing the message both internally and externally, preventing misinformation. The Legal Counsel should also be engaged early, especially if sensitive data may be involved, but the first two are critical for immediate technical and communication control.

Answers

Answer for Exercise 1

1. Physically Isolate the System: Immediately disconnect the network cable and disable Wi-Fi. This is the fastest and most effective way to prevent the potential malware from spreading to other systems on the network.

2. Notify the Incident Response Team (IRT) / Security Team: Report the incident through official channels. This activates the formal incident response process, ensuring the event is properly tracked, managed, and escalated.

3. Do Not Turn Off or Use the Computer: Leave the machine powered on but isolated. Shutting it down could erase volatile memory (RAM) which may contain crucial forensic evidence. Do not attempt to log in or "fix" it, as this could alter the state of the system and hamper investigation.

Answer for Exercise 2

Technical Control: Immutable Backups. Implement a backup solution following the 3-2-1 rule, with one copy stored on immutable storage (e.g., in the cloud with object lock or on WORM - Write-Once, Read-Many - media). This ensures that even if the live network is compromised by ransomware, the backup copy cannot be encrypted or deleted by the attacker, guaranteeing a clean source for recovery.

Administrative Control: Mandatory Phishing Awareness Training. Implement a continuous security training program that includes simulated phishing campaigns. This educates employees on how to spot and report malicious emails, which are the primary delivery vector for ransomware. The policy would mandate participation and track performance, reducing the likelihood of human error leading to an initial infection.

Synergy: The training reduces the probability of a successful attack, while the immutable backups provide a robust safety net to ensure business continuity if an attack does get through. This layered approach addresses both the human and technical vulnerabilities.

Answer for Exercise 3

As the Incident Commander, my first three contacts and instructions would be:

1. Technical Lead: "We have a suspected defacement on web server [Server Name/IP]. I need you to immediately verify the compromise, preserve volatile data for forensic analysis, and begin containment procedures by isolating the server in a quarantine VLAN. Do not wipe the server yet; focus on containment and evidence preservation."

2. Communications Lead: "Our public-facing web server has been defaced. Prepare internal and external communication drafts. Internally, we need to inform key IT and management stakeholders. Externally, we need a holding statement for the website and potential media inquiries. All communications must be approved by me before release."

3. Legal Counsel: "I am activating the IRT for a security incident involving a public web server. Please be on standby to advise on potential data breach notification requirements and evidence handling procedures. We are currently in the containment and investigation phase."
