CRITICAL WARNING: Read Before Using Any Security Tool

The tools discussed are powerful and for legitimate, defensive purposes. The only thing that separates a professional security researcher from a criminal is **explicit, written permission**.

Using these tools improperly can have severe, life-altering consequences. Before you proceed, you must understand the following.

The Law in Canada: Criminal Code

Unauthorized use of security tools is a federal crime governed by the **Criminal Code of Canada**.

Section 342.1: Unauthorized use of a computer

Makes it illegal to access a computer service "without color of right" (without legal authority). Using a tool like Metasploit without permission is a direct violation.

Section 430(1.1): Mischief in relation to data

Makes it a crime to willfully destroy, alter, or interfere with data. Accidentally crashing a server during an unauthorized scan can lead to charges.

Consequences are severe:

  • Criminal charges & prison (up to 10 years)
  • Civil lawsuits for damages
  • Termination from employment/expulsion
  • A ruined career in cybersecurity
