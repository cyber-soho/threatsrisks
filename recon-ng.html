What is Reconnaissance?

Reconnaissance is the first and most critical phase of any penetration test. It is the art of gathering information to build a comprehensive profile of a target's digital footprint. The more you know, the higher your chance of success. This section introduces the foundational concepts of recon and the powerful tool designed to master it: Recon-ng.

Passive Reconnaissance

Collecting information from publicly available sources without directly interacting with the target. This method is stealthy and goes unnoticed.

Examples: Searching Google, inspecting WHOIS records, browsing social media.

Active Reconnaissance

Involves direct interaction with the target's infrastructure. This can yield more detail but is "noisier" and can be detected by security systems.

Examples: Port scanning, banner grabbing, probing web servers.

Installation & Setup

This section guides you through getting Recon-ng running. We'll cover prerequisites and the recommended installation process on Kali Linux. A proper setup is the first step towards organized and effective reconnaissance missions.

Installation on Kali Linux (Recommended)

On a modern Kali system, Recon-ng is often pre-installed. If not, the process is straightforward. Open a terminal and run the following commands:

# Update your package list

$ sudo apt update

# Install Recon-ng

$ sudo apt install recon-ng

# Launch Recon-ng

$ recon-ng

Understanding Workspaces

A workspace is a self-contained environment for a specific engagement, storing all collected data in its own database file. This is crucial for keeping projects organized. Always create a new workspace for each target.

workspaces create <name>
workspaces select <name>
workspaces list
workspaces remove <name>
