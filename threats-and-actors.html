🎭 Threats and Actors

🔒 Internal Threats

Types

😈 Malicious Insiders

Employees, former employees, contractors, or partners who intentionally cause harm. Motives: financial gain, revenge, ideology, sabotage.

  • 💣 Admin plants a logic bomb deleting files after termination.
  • 📂 Employee steals customer data to sell to competitor.
  • 💰 Disgruntled employee embezzles funds.

😅 Negligent Insiders

Unintentionally cause security breaches due to carelessness.

  • 🔑 Uses weak password like "password123".
  • 📧 Clicks on phishing link, downloads malware.
  • 💻 Leaves workstation unlocked.

🎭 Compromised Insiders

Accounts/devices taken over by attackers.

  • 🔓 Stolen credentials abused.
  • 🖥️ Laptop hijacked into botnet.
  • ⌨️ Keylogger installed.
Indicators
  • 👀 Unusual logins or data access
  • 📤 Data exfiltration attempts
  • 🚫 Policy violations
  • ✉️ Suspicious communications
  • ⚙️ Unauthorized system changes
  • ❌ Repeated failed logins
  • 😡 Behavior changes

🌐 External Threats

Types
  • 🦠 Malware: viruses, worms, ransomware
  • 🎭 Phishing: deception to get data
  • 🌊 DDoS: flood systems with traffic
  • 💾 SQL Injection: exploit database inputs
  • ⚡ XSS: inject scripts into pages
  • 🎯 APTs: long-term campaigns
  • 🔗 Supply Chain: compromise vendor updates
Motivations
  • 💵 Financial Gain
  • 🕵️ Espionage
  • ⚔️ Cyber Warfare
  • ✊ Hacktivism
  • 😠 Revenge
  • 🤓 Curiosity / Challenge
